# **App Name**: WealthPath Courses Test

## Core Features:

- Quiz Presentation: Display multiple-choice questions in a clear, sectioned format, drawn from the provided questions.
- Answer Submission: Allow users to select an answer for each question.
- Score Calculation: Automatically calculate the user's score based on correct answers.
- Results Display: Show the user their score, with potential breakdowns by section.
- Explanation of Correct Answers: Optionally, provide a brief explanation of why each answer is correct (requires manual input of explanations).
- Celebration Effects: Add visual and/or audio celebration effects for every correct answer.

## Style Guidelines:

- Primary color: Lime green (#32CD32) to convey growth and learning.
- Background color: Deep navy (#00308F) for a modern, sophisticated feel.
- Accent color: Baby blue (#87CEEB) for interactive elements and highlights.
- Body and headline font: 'Inter' (sans-serif) for a modern, clean, and easily readable experience. Note: currently only Google Fonts are supported.
- Clean, sectioned layout with clear visual separation between questions and answer choices.
- Use simple, intuitive icons related to finance and learning to enhance the user experience.
- Subtle animations to provide feedback on quiz submissions and results.
